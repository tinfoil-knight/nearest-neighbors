# nearest-neighbors

Collection of algorithms to find the nearest neighbors in a vector dataset.

## Implementations
- [k-D Tree](https://en.wikipedia.org/wiki/K-d_tree)
- [Vantage-Point / VP Tree](https://en.wikipedia.org/wiki/Vantage-point_tree)
